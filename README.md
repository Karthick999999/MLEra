# MLera Linear Regression

A Next.js 14 application (React 18) built with Yarn. Includes Tailwind CSS and shadcn/ui-style components for a simple, modern UI.

## Tech Stack
- Next.js 14
- React 18
- Tailwind CSS 3
- Yarn 1 (Classic)

## Getting Started
1. Install dependencies
```bash
yarn install
```
2. Run the development server
```bash
yarn dev
```
- App runs at: http://localhost:3000

3. Build for production
```bash
yarn build
```
4. Start production server
```bash
yarn start
```

## Scripts
- `yarn dev` – Start local dev server
- `yarn build` – Build the app
- `yarn start` – Run the built app
- `yarn lint` – Lint the project

## Project Notes
- Node.js 18+ recommended (project tested with Node v22.x on Windows)
- Package manager: Yarn 1.x (yarn.lock present)
